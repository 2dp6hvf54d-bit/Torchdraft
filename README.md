# TorchDraft v0.4

Fungerande alpha för CNC-plasma:

- Skyltgenerator
- Snabbmall för husnummer
- Snabbmall för eldkorgspanel
- Snabbmall för väggkonst

- DXF-export
- SVG-export
- Spara och öppna senaste projekt
- Enkel Plasma Check
- Mobilanpassning

Kontrollera alltid exporterade filer i ArcDroid före skärning.
