# TorchDraft

TorchDraft är en webbaserad app för personer som vill skapa CNC-plasmaskärfiler utan att kunna CAD.

## Alpha v0.3
- Startsida med enkla val
- Fungerande skyltgenerator
- Mått i millimeter
- Förhandsgranskning
- Monteringshål
- DXF-export
- Spara projekt
- Feedbackfunktion
- Mobilanpassad

## Nästa steg
- Fyllda bokstavskonturer
- Automatiska bryggor
- Plasma Check
- Husnummer
- AI Pro
